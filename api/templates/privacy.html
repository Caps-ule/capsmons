CapsMöns

POLITIQUE DE CONFIDENTIALITÉ — TWITCH EXTENSION

1. Qui sommes-nous ?

CapsMöns est une extension Twitch développée pour la chaîne capsloque. Le service est opéré par le streamer capsloque et hébergé sur capsmons.devlooping.fr.

2. Données collectées

Lorsque vous utilisez le panneau CapsMöns et activez le partage d'identité Twitch (Identity Linking), nous accédons aux données suivantes :

— Votre identifiant Twitch (user_id), fourni par Twitch via le mécanisme d'extension JWT.
— Votre nom d'utilisateur Twitch (login), récupéré lors de votre première connexion au site capsmons.devlooping.fr.

Ces données sont utilisées uniquement pour associer votre profil de jeu (compagnons, inventaire, progression XP) à votre compte Twitch.

3. Ce que nous ne collectons pas

Nous ne collectons pas d'adresse e-mail, de données de paiement, de localisation, ni aucune autre information personnelle. Nous n'utilisons pas Google Analytics ni aucun outil de tracking tiers.

4. Utilisation des données

Vos données sont utilisées exclusivement pour :

— Afficher votre compagnon actif, votre collection et votre inventaire dans le panneau.
— Enregistrer vos actions (changement de CM actif, utilisation d'objets).
— Suivre votre progression dans le jeu CapsMöns.

Vos données ne sont jamais vendues, partagées ou transmises à des tiers.

5. Conservation des données

Vos données sont conservées tant que votre profil est actif sur CapsMöns. Vous pouvez demander la suppression de vos données à tout moment en contactant capsloque sur Twitch.

6. Sécurité

Les communications entre le panneau et notre serveur sont chiffrées via HTTPS. L'authentification repose sur les JWT signés par Twitch — nous ne stockons aucun mot de passe.

7. Contact

Pour toute question relative à vos données, contactez-nous via Twitch : twitch.tv/capsloque.
